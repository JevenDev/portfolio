# JVN Portfolio (Vue + Tailwind + GSAP)

Minimal portfolio rebuild using Vue 3 (Composition API), Tailwind CSS, and GSAP.

## Install

```bash
npm install
```

## Run locally

```bash
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Content source of truth

- `data/config.json`
- `data/projects.json`
- `data/artists.json`
- Static media: `public/assets/**`
